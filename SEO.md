## Implementation Steps

### 1. Add Site URL to Configuration
**File:** `astro.config.mjs`
```javascript
export default defineConfig({
  site: 'https://cscs.dev', // Replace with actual domain
  // ... rest of config
});
```

### 2. Install and Configure Sitemap
**Command:**
```bash
npm install @astrojs/sitemap
```

**File:** `astro.config.mjs`
```javascript
import sitemap from '@astrojs/sitemap';

export default defineConfig({
  site: 'https://cscs.dev',
  integrations: [react(), sitemap()]
});
```

This automatically generates `/sitemap-index.xml` and `/sitemap-0.xml` at build time.

### 3. Create robots.txt
**File:** `public/robots.txt`
```
User-agent: *
Allow: /

Sitemap: https://cscs.dev/sitemap-index.xml
```

### 4. Enhance Layout.astro with SEO Meta Tags

**Add to Layout.astro:**
- Canonical URL
- Open Graph tags (title, description, image, type, url)
- Twitter Card tags
- Additional meta tags (author, keywords)

**New Props:**
```typescript
interface Props {
  title?: string;
  description?: string;
  image?: string; // For OG/Twitter cards
  type?: 'website' | 'article'; // OG type
  keywords?: string[]; // Optional keywords
}
```

### 5. Add Structured Data (JSON-LD)

**Organization Schema (Layout.astro):**
```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "College Station Computer Science",
  "url": "https://cscs.dev",
  "logo": "https://cscs.dev/logo.png",
  "description": "...",
  "sameAs": [
    // Social media URLs when available
  ]
}
```

**Blog Post Schema (blog post layout):**
```json
{
  "@context": "https://schema.org",
  "@type": "BlogPosting",
  "headline": "Post Title",
  "image": "...",
  "datePublished": "...",
  "dateModified": "...",
  "author": {
    "@type": "Person",
    "name": "Author Name"
  },
  "publisher": {
    "@type": "Organization",
    "name": "College Station Computer Science",
    "logo": {
      "@type": "ImageObject",
      "url": "..."
    }
  },
  "description": "..."
}
```

### 6. Update Blog Post Pages

**File:** `src/pages/blog/[slug].astro`
- Pass blog post metadata to Layout (image, type: 'article')
- Add structured data script
- Ensure proper heading hierarchy

### 7. Create Blog Post Layout (Optional)

**File:** `src/layouts/BlogLayout.astro`
- Extends Layout.astro
- Adds article-specific structured data
- Adds article-specific meta tags
- Provides consistent blog post structure
